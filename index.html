CentOS System Preparation Guide

Overview

Prepare a CentOS 7.9 system for enterprise platform deployment. Complete all preparation tasks before proceeding with installation.

Task 1: System Prerequisites

  • CentOS 7.9 minimal installation
  • Minimum 16GB RAM, 8 CPU cores
  • 500GB+ storage space
  • Network connectivity
  • Root access

Task 2: Configure Network & Firewall

Open required ports for platform communication:

# Open platform ports
firewall-cmd --permanent --add-port=22/tcp     # SSH
firewall-cmd --permanent --add-port=80/tcp     # HTTP
firewall-cmd --permanent --add-port=443/tcp    # HTTPS
firewall-cmd --permanent --add-port=8443/tcp   # Management
firewall-cmd --permanent --add-port=5432/tcp   # Database
firewall-cmd --permanent --add-port=9200/tcp   # Elasticsearch
firewall-cmd --permanent --add-port=6443/tcp   # Kubernetes API
firewall-cmd --reload

Task 3: Create & Configure Swap

Platform requires adequate swap space:

# Create 8GB swap file
dd if=/dev/zero of=/swapfile bs=1024 count=8388608
chmod 600 /swapfile
mkswap /swapfile
swapon /swapfile

# Make permanent
echo '/swapfile swap swap defaults 0 0' >> /etc/fstab

Task 4: Configure NTP

Ensure accurate time synchronization:

yum install -y ntp
systemctl enable ntpd
systemctl start ntpd

# Configure NTP servers in /etc/ntp.conf
server 0.centos.pool.ntp.org
server 1.centos.pool.ntp.org

Task 5: Configure YUM Proxy

Setup proxy for package management:

# Add to /etc/yum.conf
proxy=http://proxy.company.com:8080
proxy_username=your_username
proxy_password=your_password

Task 6: Install Required Packages

Install system dependencies:

yum update -y
yum install -y wget curl unzip tar net-tools
yum install -y device-mapper-persistent-data lvm2
yum install -y yum-utils

Task 7: Configure Platform Settings

Edit the platform configuration file for single-node deployment with separate database host.

Configuration file: /opt/platform/config/platform-config.yaml

Configuration Templates

  • platform-config.yaml - Main platform configuration
  • database-config.yaml - Database connection settings
  • ntp.conf.template - NTP configuration template

Available Commands

System Commands:

  • ls [-la] - List files
  • cd [dir] - Change directory
  • cat [file] - View file contents
  • vi [file] - Edit file
  • pwd - Show current directory
  • clear - Clear terminal

System Administration:

  • systemctl [action] [service] - Manage services
  • firewall-cmd [options] - Configure firewall
  • yum [command] - Package management
  • free -h - Check memory/swap
  • df -h - Check disk space

Network Commands:

  • netstat -tuln - Show listening ports
  • ss -tuln - Show socket statistics
  • ping [host] - Test connectivity
